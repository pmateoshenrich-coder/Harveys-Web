# Harvey's Cocktail Bar

Sitio web del speakeasy **Harvey's · The Invisible Bar** (Calle de San Lorenzo, 5 · Madrid Centro).

Web estática (HTML + CSS, sin build) optimizada para SEO, lista para desplegar en Vercel.

## Páginas

- `index.html` — Inicio
- `carta.html` — Carta de cócteles
- `sobre-harveys.html` — Sobre Harvey's y Eduardo Gutiérrez
- `donde-estamos.html` — Ubicación y cómo llegar
- `reservar-grupos.html` — Reservas y privatización de grupos

## Estructura

- `img/` — Imágenes optimizadas en formato WebP
- `sitemap.xml`, `robots.txt` — SEO
- `vercel.json` — URLs limpias (`cleanUrls`, `trailingSlash`)

## Despliegue

El proyecto es estático: Vercel lo sirve directamente sin paso de build.
Cada `push` a la rama principal despliega automáticamente.
